# mini-projects

Mini bioinformatics projects exploring statistical modeling and transcriptomics analysis.

## Projects
- **mini_logreg**: Logistic regression (scikit-learn) for breast tumor classification + ROC/AUC + coefficient interpretation.
- **Spatial_Transcriptomics**: Simple spatial transcriptomics visualization.

## Skills demonstrated
- Python, scikit-learn, pandas, matplotlib
- Model evaluation (train/test split, ROC/AUC)
- Interpretation of coefficients and biological context

## Notes
These projects are learning-focused and emphasize connecting statistical modeling to biological interpretation.
